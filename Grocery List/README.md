## 🛍 Grocery Goods List
## 📌 Overview
***Grocery Goods*** is a functional, browser-based **To-Do/Grocery List application** built with **HTML, CSS, and Vanilla JavaScript**. It enables users to efficiently manage their shopping lists with CRUD (Create, Read, Update, Delete) functionality and persistent data storage.

---

## ✨ Features
* **Dynamic CRUD Operations -** Users can easily add new items, edit existing ones, or remove them from the list.
* **Persistent Storage -** Leverages `localStorage` so that your list remains saved in your browser even after refreshing or closing the page.
* **Interactive Feedback -** Provides real-time alerts (success/danger) when adding, editing, or clearing items.
* **Responsive & Clean UI -** A minimalist design that is easy to read and intuitive to use on any screen size.
* **Bulk Clear -** A quick-action button to wipe the entire list at once.

---

## 🚀 Getting Started
1. Clone or download this repository.
2. Open `index.html` in your browser to begin managing your list.
