## 🛒 Amazon UI Clone
## 📌 Overview
***Amazon UI Clone*** is a responsive front-end project that replicates the layout and design of the Amazon homepage. This project focuses on mastering CSS Flexbox, 3D structure, and clean component-based layout design using **HTML and CSS**.

---

## ✨ Features

* **Responsive Navigation Bar -** A multi-level header featuring search inputs, location selection, and account/cart menus.
* **Hero Section -** A promotional hero banner with a centered message overlay.
* **Product Grid -** A card-based layout system to showcase various product categories.
* **Complex Footer -** A detailed multi-column footer architecture replicating the actual Amazon user experience.
* **Modern CSS Techniques -** Uses `Flexbox`, `CSS Variables` (in spirit), and hover-state effects for a professional look.

---

## 🚀 Getting Started

1. Clone or download this repository.
2. Ensure you have an `images/` folder in the root directory containing the relevant assets (logo, hero image, and product box images).
3. Open `index.html` in your browser to view the interface.
